# PayShield

**PayShield** is a real-time fraud detection and decision intelligence platform designed for high-impact fraud prevention *before* transaction execution.

## Features
- **Real-time Risk Scoring**: Instant analysis of transaction data.
- **Behavioral DNA**: Identity verification through typing and movement analysis.
- **Graph Intelligence**: Detection of fraud rings and risky network connections.
- **Explainable AI**: Transparent decision rationale for every transaction.
- **Adaptive Policy**: Dynamic responses ranging from ALLOW to BLOCK.

## Architecture
- **Frontend**: Next.js + Tailwind CSS + Framer Motion.
- **Logic**: Local simulation engine for high-performance intelligence demonstration.
- **Data**: JSON-based intelligence layers.

## Getting Started
To run the project locally:
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
